How does an image look after we run some simple convs?

Using convolutions, we can construct a "network" of convolutions that feed into each other; i.e. convolving an image that's already been convolved once. Some of these convolutions give great analysis of the images as-is, why string multiple together?

We would like our computer to guess what it is seeing accurately as possible. Go ahead and adjust the slider below and try and help the model correctly predict this dog. By adjusting the slider, you are changing the values within the image convolutions.

Now we have a network that can correctly identify this dog! However, it would be nice to have our network adjust the slider automatically without our help. Luckily, there is a technique called "Back-propagation"(link) that works similarly to what you did when adjusting the slider! The computer will make some predictions and it will move the slider in the direction that seems to best predict the dog.

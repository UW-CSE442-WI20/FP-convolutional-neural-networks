How does an image look after we run some simple convs?

Using convolutions, we can construct a "network" of convolutions that feed into each other; i.e. convolving an image that's already been convolved once. Some of these convolutions give great analysis of the images as-is, why string multiple together?

Below is a simple neural network for classifying a picture as one of ten categories: Plane, Car, Bird, Car, Deer, Dog, Frog, Horse, Ship and Truck. Hover over the pixels in any layer to see the path of a pixel through the network. Both the relevant pixel and the kernel are highlighted in alternating colors at each layer.

On the right the top 3 predicted probabilities of each class are displayed. Right now the network predicts the class of the image horribly wrong. Use the slider to adjust the weights of the network so that the network predicts the image to be a dog with 100% probability and everything else with 0% probability. As you adjust the slider, watch the networks prediction and note how the output of the first convolution becomes more and more similar the convolved images we saw earlier. This means the network is learning useful kernels.

Once you are done, try selecting some of the other images above!

In reality, we don't tune the networks manually like this. Most commonly an algorithm called Backpropagation is used. It uses the network to predict an image and then based on how wrong the prediction is, the weights of the network are adjusted so that the network predicts the image correctly. Through this process, the network is able to automatically learn complex kernels that extract useful information from images.

Through backpropagation, we trained a convolutional neural network to predict an images as one of the 10 classes. This small convolutional neural network is run directly in your browser. Below are 32 images from each of the 10 classes that the network has never seen before. Click on any image to select it and have the network predict what the image is.
